## Get-Started

```bash
git clone https://github.com/wl00887404/Apollo_Get_Started
cd Apollo_Get_Started
git checkout 6701ffae5613f7aef43857511e911fdb31261344
npm install 
```
### [Server](http://localhost:8081/)
#### Port: 8081

```bash
npm run server
```

### [Client](http://localhost:8080/)
#### Port: 8080

```bash
npm run start
```
